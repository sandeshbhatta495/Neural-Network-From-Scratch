# Neural Networks From Scratch

> A comprehensive exploration of Neural Networks based on **"Neural Networks from Scratch in Python"** by Harrison Kinsley and Daniel Kukiela

---

## üìö About This Project

This project implements neural networks from **first principles** without relying solely on high-level frameworks. Each chapter provides hands-on implementations and deep understanding of how neural networks actually work at the mathematical and computational level.

**Primary Reference**: Neural Networks from Scratch in Python

---

## üéØ What You'll Learn

### **Core Concepts**
- Mathematical foundations of neural networks
- Forward and backward propagation algorithms
- Activation functions and their role in non-linearity
- Loss functions and their applications
- Optimization algorithms (Gradient Descent, SGD, Advanced Optimizers)
- Regularization techniques (L1/L2, Dropout)
- Batch normalization and training improvements

### **Model Architectures**
- Dense/Fully-Connected Neural Networks
- Convolutional Neural Networks (CNNs)
- Recurrent Neural Networks (RNNs) and LSTMs
- Advanced architectures and design patterns

### **Practical Skills**
- Implement layers, activation functions, and loss functions from scratch
- Build complete training pipelines
- Model evaluation and validation strategies
- Model saving, loading, and inference
- Working with real datasets
- Binary classification and regression
